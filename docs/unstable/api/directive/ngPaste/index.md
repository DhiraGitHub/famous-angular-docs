<h1 class="api-title">

  ng-paste



</h1>





Specify custom behavior on paste event.






  
<h2 id="usage">Usage</h2>
  
    ```html
  <window, input, select, textarea, a
    ng-paste="">
  ...
  </window, input, select, textarea, a>
  ```
