<h1 class="api-title">

  fa-pipe-from



</h1>





This directive remove an handler object from set of downstream handlers. Undoes work of "pipe"
from a faPipeTo directive.








  
<h2 id="usage">Usage</h2>
  
```html
<ANY fa-pipe-from="EventHandler">
  <!-- zero or more render nodes -->
</ANY>
```
